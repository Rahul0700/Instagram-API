# Instagram-API

Instagram back-end API constructed using Go lang and MongoDB - Task at APPOINTY INTERNSHIP PROGRAM

###### To develop locally

```
git clone https://github.com/Rahul0700/Instagram-API.git

cd Instagram-API/src

go install

go build
```

###### In Windows

```
src.exe
```

###### In Mac/Linux

```
./src
```
